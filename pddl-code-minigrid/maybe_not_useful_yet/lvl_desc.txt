    GoToRedBallNoDists: Navigate to a red ball without any distractors.

    GoToObj: Navigate to an object specified by type and color.

    GoToLocal: Navigate to an object located nearby.

    GoTo: Navigate to an object that may be in a different room.

    GoToImpUnlock: Navigate to an object, unlocking doors as needed.

    GoToSeq: Navigate to multiple objects in a specified sequence.

    GoToDoor: Navigate to a specified door.

    GoToObjDoor: Navigate to a door specified by object properties.
